# 三、我直接给你【第一个可上线工具代码】
## 工具名称：**长辈用药提醒（纯前端本地存储）**
你复制就能用，AI 也能帮你美化。

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>长辈用药提醒 - 大字版</title>
  <style>
    * { font-size: 28px; box-sizing: border-box; }
    body { padding: 20px; background: #f9f9f9; }
    .container { max-width: 500px; margin: 0 auto; }
    .btn { 
      width: 100%; 
      padding: 20px; 
      margin: 10px 0; 
      background: #42b983; 
      color: white; 
      border: none; 
      border-radius: 10px;
    }
    input {
      width: 100%;
      padding: 18px;
      margin: 10px 0;
      border-radius: 8px;
      border:1px solid #ddd;
    }
    .item {
      background: white;
      padding: 20px;
      margin: 10px 0;
      border-radius: 10px;
    }
  </style>
</head>
<body>
<div class="container">
  <h1>💊 长辈用药提醒</h1>
  <input type="text" id="name" placeholder="药名（如：降压药）">
  <input type="text" id="time" placeholder="时间（如：早8点）">
  <button class="btn" onclick="add()">添加提醒</button>
  <div id="list"></div>
</div>

<script>
// 本地存储，不用数据库
let list = JSON.parse(localStorage.getItem('meds')) || []
render()

function add() {
  let name = document.getElementById('name').value
  let time = document.getElementById('time').value
  if (!name || !time) return alert('请填写完整')
  list.push({name, time})
  localStorage.setItem('meds', JSON.stringify(list))
  document.getElementById('name').value = ''
  document.getElementById('time').value = ''
  render()
}

function render() {
  let html = ''
  list.forEach((i, idx) => {
    html += `
      <div class="item">
        ${idx+1}、${i.name} <br> ⏰ ${i.time}
        <button onclick="del(${idx})" style="background:red;color:white;padding:10px;margin-top:10px;">删除</button>
      </div>
    `
  })
  document.getElementById('list').innerHTML = html
}

function del(idx) {
  list.splice(idx, 1)
  localStorage.setItem('meds', JSON.stringify(list))
  render()
}
</script>
</body>
</html>
```

## 功能：
- 添加药名 + 时间
- 本地保存（关闭网页还在）
- 删除
- 超大字体、适合老人

**你只需要美化样式，就能上线。**
