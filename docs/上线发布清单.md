## 3. Vercel 上传步骤
推荐两种方式，优先选一种即可。

### 方式一：Git 导入
1. 先把项目上传到 GitHub
2. 打开 [Vercel](https://vercel.com/)
3. 登录账号
4. 点击 `Add New...`
5. 选择 `Project`
6. 在 `Import Git Repository` 里选择你的仓库
7. 保持静态站点默认配置
8. 点击部署

### 方式二：Vercel CLI
1. 安装 Vercel CLI：`npm i -g vercel`
2. 打开终端，进入项目目录 `/Users/zxx/Desktop/yin_fa`
3. 执行：`vercel --prod`
4. 根据提示完成首次项目绑定
5. 等待生成正式链接

## 4. Vercel 配置建议
- Framework Preset：保持默认或选择 `Other`
- Build Command：留空
- Output Directory：留空
- Install Command：留空

说明：
- 当前项目是纯静态站点，不需要 Node 构建流程
- `vercel.json` 已处理首页和 4 个工具页的访问路径
- 根据 Vercel 官方文档，静态 HTML/CSS/JS 项目可以不设置构建命令
